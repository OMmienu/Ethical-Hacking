# BeEF Tabnabbing Attack

## Objective
To demonstrate how inactive browser tabs can be exploited to steal sensitive user information.

## Steps
1. Launch BeEF and hook the victim browser
2. Navigate to:
   Command → Social Engineering → Tabnabbing
3. Set wait time to 1 minute
4. Victim returns and types sensitive information

## Observation
Entered text is logged inside the BeEF control panel.

## Lesson Learned
Users should avoid entering sensitive data into tabs left unattended for long periods.
