# Ethical Hacking Labs Documentation

## Overview
This repository documents two ethical hacking practical labs conducted in a controlled environment for **educational and penetration testing learning purposes only**.

### Labs Covered
1. Website Cloning & Browser Exploitation (BeEF & SEToolkit)
2. SMB Vulnerability Scanning using Enum4Linux & smbclient

⚠️ **Ethical Disclaimer**  
All activities were performed in an isolated lab environment. No real users, production systems, or unauthorized networks were targeted.
